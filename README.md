# Toxic-classifier NLP
A lightweight NLP project that classifies toxic comments using a 3-step text cleaning pipeline, 1 TF-IDF vectorizer, and 1 ML model. The classifier predicts 6 toxicity categories and achieves strong performance with accuracy, precision, recall, and F1-score metrics. Built using Python, Pandas, NLTK, and Scikit-learn, and saved as Toxic_analyzer.pkl.
